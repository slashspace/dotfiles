# aerospace

## Focus
alt + h    Focus window left
alt + j    Focus window down
alt + k    Focus window up
alt + l    Focus window right

## Move
alt + shift + h    Move window left
alt + shift + j    Move window down
alt + shift + k    Move window up
alt + shift + l    Move window right

## Workspaces
alt + 1    Switch to workspace 1
alt + 2    Switch to workspace 2
alt + 3    Switch to workspace 3
alt + 4    Switch to workspace 4
alt + 5    Switch to workspace 5
alt + 6    Switch to workspace 6
alt + 7    Switch to workspace 7
alt + 8    Switch to workspace 8
alt + 9    Switch to workspace 9
alt + 0    Switch to workspace 0
alt + tab    Toggle last two workspaces

## Move to Workspace
alt + shift + 1    Move window to workspace 1
alt + shift + 2    Move window to workspace 2
alt + shift + 3    Move window to workspace 3
alt + shift + 4    Move window to workspace 4
alt + shift + 5    Move window to workspace 5
alt + shift + 6    Move window to workspace 6
alt + shift + 7    Move window to workspace 7
alt + shift + 8    Move window to workspace 8
alt + shift + 9    Move window to workspace 9
alt + shift + 0    Move window to workspace 0
alt + shift + tab  Move workspace to next monitor

## Layout
alt + /      Toggle tiles layout (horizontal/vertical)
alt + ,      Toggle accordion layout (horizontal/vertical)

## Resize Mode (alt+shift+r to enter)
h    Resize width -50
j    Resize height +50
k    Resize height -50
l    Resize width +50
b    Balance sizes
-    Smart resize -50
=    Smart resize +50
enter / esc  Return to main mode

## Service Mode (alt+shift+; to enter)
esc    Reload config
r      Flatten workspace tree (reset layout)
f      Toggle floating/tiling for current window
backspace  Close all windows except current
alt+shift+h/j/k/l  Join window with neighbor
