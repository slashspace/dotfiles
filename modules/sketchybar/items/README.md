# SketchyBar 模块分类

模块由主配置通过 `load_items.sh` 统一加载。

## Active（当前启用）

- `apple.sh` — 系统菜单（偏好、活动、锁屏）
- `spaces.sh` — AeroSpace 工作区条（0–9、当前工作区应用）
- `spotify.sh` — Spotify 状态
- `calendar.sh` — 日期时间
- `brew.sh` — Homebrew 更新
- `dnd.sh` — 勿扰模式
- `github.sh` — GitHub 通知
- `wifi.sh` — Wi‑Fi 状态
- `battery.sh` — 电池
- `volume.sh` — 音量（右击切换输出）
