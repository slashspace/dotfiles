# ghostty

## Config
shift + cmd + ,    Reload config (default)
super + b > r      Reload config (custom)
super + i          Toggle inspector

## Quick Terminal
super + b > ,    Toggle quick terminal

## Tabs & Windows
super + b > c    New tab
super + b > n    New window
super + b > x    Close surface (tab/split)
super + b > 1-9  Go to tab 1-9

## Splits
super + b > \    Split right
super + b > -    Split down
super + b > e    Equalize splits

## Split Navigation
super + b > h    Go to split left
super + b > j    Go to split bottom
super + b > k    Go to split top
super + b > l    Go to split right
