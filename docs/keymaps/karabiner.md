# karabiner

## CapsLock Remapping
caps_lock (hold)    Hyper key (Cmd+Ctrl+Alt+Shift)
caps_lock (alone)   Escape

## Hyper Navigation (Vim-style)
hyper + h    Arrow left
hyper + j    Arrow down
hyper + k    Arrow up
hyper + l    Arrow right
hyper + ;    Cmd + arrow right (end of line)

## Vim Navigation (Alternate)
hyper + j    Arrow left (jikl layout)
hyper + i    Arrow up (jikl layout)
hyper + k    Arrow down (jikl layout)
hyper + l    Arrow right (jikl layout)

## Ctrl Navigation
ctrl + h    Arrow left
ctrl + j    Arrow down
ctrl + k    Arrow up
ctrl + l    Arrow right

## CapsLock Toggle
left_shift + right_shift    Toggle caps_lock

## Shift Word Navigation
right_shift (alone)    Move forward one word (option+right)
left_shift (alone)     Move backward one word (option+left)

## App Launcher (o + key)
o + n    Open Notion
o + m    Open Mail
o + s    Open Spotify
o + b    Open Brave Browser
o + a    Open Activity Monitor
o + c    Open VS Code
o + i    Open iTerm

## System
hyper + space    Toggle Alfred (cmd+space)
cmd + w          Disabled (no accidental tab close)
cmd + q          Double-tap required to quit apps
print_screen     Open Screenshot.app
